Task 3. Express.js Framework: Dive into Express.js, a popular web application framework for
Node.js. Learn how to set up an Express project, handle routes, and use middleware effectively.
Understand concepts like request-response cycle, routing, and error handling. Use of json files
for controlling server or service related variables. Use nodemon for rapid development.
